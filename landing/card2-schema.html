Shell command from text to JSON with schema

echo 'my name is jay i am 30 yo' | jt --schema json_schema.json

Transforms raw text to JSON using a specified schema. For example, the above command produces:

{
    "name": "jay",
    "age": 30
}

The schema used:

{
    "$schema": "http://json-schema.org/draft-07/schema#",
    "type": "object",
    "properties": {
        "name": {
            "type": "string"
        },
        "age": {
            "type": "integer"
        }
    },
    "required": [
        "name",
        "age"
    ]
}

To install the tool, use:

pip install jsonthat
