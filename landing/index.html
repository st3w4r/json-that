JSON THAT

Text-to-json CLI Tool

Transform textual content into JSON format using this command-line interface tool, powered by large language models.

Installation

pip install jsonthat

Start by setting up your LLM provider and enter your API key when prompted:

jt --setup

Display usage

jt -h

Usage

To process text into JSON, use:

echo "Your sample text" | jt echo "Your sample text" | jt --schema json_schema.json 
$ echo 'my name is jay' | jt

{
    "name": "Jay"
}

Features

  • Supports json schema to guide outputs
  • Pipe content directly
  • Multi-LLM provider: openai, claude

Configuration

Config file

Configurations are managed through:

~/.config/jsonthat/config.yaml

Ensure your provider details and API key are set here.

Environment Variables

To configure the CLI tool, you can set the following environment variables:

export LLM_PROVIDER='openai'
export LLM_API_KEY='your_api_key_here'

Specifies the large language model provider (e.g., 'openai' or 'claude') and API key for the specified LLM provider.
